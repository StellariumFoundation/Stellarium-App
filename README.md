## ğŸ“¥ Installation Guide

We provide optimized builds for maximum performance across the Android ecosystem.

### Step 1: Choose Your Version
Go to the **[Releases Section](../../releases/latest)** and download the correct file:

| File Name | Best For | Description |
| :--- | :--- | :--- |
| `Stellarium-Universal.apk` | **Everybody** | Contains libraries for all devices. Larger file size, but guaranteed to work. |
| `Stellarium-ARM64.apk` | **Modern Phones** | Optimized for Pixel 6/7/8, Samsung S20+, and most phones bought after 2019. |
| `Stellarium-ARMv7.apk` | **Older Devices** | For budget devices or older hardware. |

### Step 2: Install
1.  Save the `.apk` file to your phone.
2.  Tap the notification when the download finishes.
3.  If prompted, go to **Settings** and enable **"Install from Unknown Sources"** for your browser.
4.  Tap **Install**.
5.  Open the app and begin your journey.
